# Message Decoder

This simple Kotlin program decodes a message using triangular numbers from a text file.

## Example

Suppose "message_file.txt" contains the following content:

1 I <br />
3 Love <br />
2 Milk <br />
5 Cheese <br />
6 Coding <br />
4 Butter <br />

The program will output the decoded message as:
`I Love Coding`
